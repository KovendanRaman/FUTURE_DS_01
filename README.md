# 📊 Business Sales Performance Analytics (FUTURE_DS_01)

## 📌 Objective
Analyze transnational e-commerce transaction data to identify revenue trends, top-performing products, and high-value regional markets.

## 💡 Key Business Insights
1. **Seasonality:** Revenue spikes significantly in Q4 (specifically November), indicating strong holiday-driven sales. 
2. **Product Concentration:** A small fraction of items drive the vast majority of revenue. 
3. **Global Expansion:** Outside of the UK market, EIRE, the Netherlands, and Germany represent the highest-value regions. 

## 🛠️ Tech Stack
* **Python:** Data manipulation and cleaning (Pandas).
* **Streamlit:** Interactive web application deployment.
* **Plotly:** Dynamic and aesthetic data visualization.

## 🚀 How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run analysis.py`
